## What is StarLake?

StarLake is a Rust-native, fully open implementation of the
[SpatioTemporal Asset Catalog (STAC)](https://stacspec.org/) API together with
a [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) layer that
makes geospatial collections directly addressable by LLM agents.

The stack is built to be **self-hosted by default**: every layer ships as a
container, a Nomad job, a Kustomize base, or an Ansible role — pick the
deployment story that matches your infrastructure.
